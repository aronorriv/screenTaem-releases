# 📦 Releases

Official download hub for our applications.

## Downloads

Head over to the [Releases](../../releases) page to download the latest versions.

## Available Platforms

| Platform | Format |
|----------|--------|
| macOS    | `.dmg` |
| Windows  | `.exe` / `.msi` |
| Linux    | `.AppImage` / `.deb` |

## Installation

1. Go to [Releases](../../releases)
2. Find the version you need
3. Download the file for your platform
4. Run the installer

## Checksums

Each release includes SHA-256 checksums to verify file integrity:

```bash
# macOS / Linux
shasum -a 256 <filename>

# Windows (PowerShell)
Get-FileHash <filename> -Algorithm SHA256
```
